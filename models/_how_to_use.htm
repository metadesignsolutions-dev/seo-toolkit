How to Use SEO Meta/OG/Schema Tags

  1. Add the SeoMeta component to your layout or page: 
    [seoMeta]
==
  2. In your <head> section, add: 
    
<title>{{ this.page.meta_title ?: seoMeta.site_title }}</title>
<meta name="description" content="{{ this.page.meta_description ?: seoMeta.site_description }}">
<meta name="keywords" content="{{ this.page.meta_keywords ?: seoMeta.meta_keywords }}">
<meta property="og:title" content="{{ this.page.og_title ?: seoMeta.og_title }}">
<meta property="og:description" content="{{ this.page.og_description ?: seoMeta.og_description }}">
<meta property="og:image" content="{{ this.page.og_image ?: seoMeta.og_image }}">
<meta property="og:type" content="{{ this.page.og_type ?: seoMeta.og_type }}">
<meta name="twitter:title" content="{{ this.page.twitter_title ?: seoMeta.twitter_title ?: this.page.og_title ?: seoMeta.og_title }}">
<meta name="twitter:description" content="{{ this.page.twitter_description ?: seoMeta.twitter_description ?: this.page.og_description ?: seoMeta.og_description }}">
<meta name="twitter:card" content="{{ this.page.twitter_card ?: seoMeta.twitter_card ?: 'summary' }}">

{% if this.page.schema_jsonld %}
{{ this.page.schema_jsonld|raw }}
{% elseif seoMeta.schema_custom_jsonld %}
{{ seoMeta.schema_custom_jsonld|raw }}
{% endif %}
  3. For page-specific tags, add fields to your page and use this.page.meta_title, etc.

Tip: If you only want global tags, just use the seoMeta variables. If you want to override per page, set the variables in your page or post.
